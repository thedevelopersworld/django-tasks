1) Create new project django-task-4
command : django-admin startproject django-task-4

2) Change directory to "django-task-4"
command : cd django-task-4

3) Create new folder static and templates

4) Go to templates folder and create new file as index.html and add some basic html code

5) Then go to static folder and create new file style.css and add some css for index.html file

6) After that go to index.html file. go to the head tag and add < link rel="stylesheet" type="text/css" href="{% static 'style.css' %}">

7) Go to settings.py then go to DIRS section and add dir as 'templates' then go to end of the file and add STATICFILES_DIRS = [ os.path.join(BASE_DIR,'static'), ]

8) Go to django-task-4 folder and create new file views.py and define a function named index which will render index.html file

9) Go to urls.py. import views by adding code in urls.py from . import views then add url path('',views.index)

10) Run manage.py file
command : python3 manage.py runserver

11) Grab screenshot of output then upload folder and screenshot of output to folder associated with your name to google drive by clicking on the Submit Code button
